This project is a personal travel utility web app.
It is built for REAL usage during trips, not as a demo or portfolio piece.

---

## PURPOSE

- Provide quick-access travel tools
- Reduce friction while traveling
- Replace multiple apps with one simple interface

---

## TARGET USAGE

- Opened on mobile phone during trips
- Used while walking, ordering, paying, or checking info
- Sessions are short and task-focused

---

## USER PROFILE

- Individual traveler or small group
- Non-technical
- Wants speed and clarity
- Does not want to learn a complex system

---

## SCOPE

INCLUDED:
- Weather
- Currency conversion
- Image-based translation
- Expense tracking and splitting

EXCLUDED:
- Authentication
- Social features
- Cloud accounts
- Admin panels
- Advanced analytics

---

## TECH ASSUMPTIONS

- Mobile-first
- Web-based (can be PWA later)
- Simple local or lightweight persistence
- No backend database unless approved
