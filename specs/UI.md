This app is utility-first.
Design must reduce thinking, not impress.

---

## CORE PRINCIPLES

- Extremely simple
- Calm and neutral
- No visual noise
- No unnecessary decoration

---

## LAYOUT RULES

- Mobile-first always
- Vertical scrolling
- One primary action per screen
- Clear section separation

---

## INTERACTION

- Large tap targets
- Clear icons WITH text labels
- Minimal steps to complete a task
- No hidden gestures

---

## COMPONENT PREFERENCES

- Prefer lists over cards
- Prefer text over graphics
- Use cards ONLY when they improve clarity

---

## AVOID

- Dashboards
- Dense information blocks
- Fancy animations
- Trendy UI patterns
- Over-styled components

---

If UI feels "boring but clear", it is correct.
